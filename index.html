04 / Proyectos

Software que shippeo.

Proyectos donde diseñé arquitectura, backend, seguridad y frontend de punta a punta.

PROYECTO DESTACADO

NexaFisio

Ver demo

Plataforma SaaS multi-sede para clínicas de kinesiología — turnos, historias clínicas, roles y almacenamiento seguro de adjuntos.

  • Arquitectura: API REST con Spring Boot 3, Spring Security y JWT, con aislamiento de datos por sede (multi-tenant vía filtros Hibernate y header X-Sede-Id).
  • Backend: Módulos de turnos (agenda, recurrencia, reprogramación), pacientes, tratamientos jerárquicos, usuarios con roles (super admin / org / sede) y soft delete con papelera de restauración.
  • Seguridad: Autenticación JWT, bloqueo por intentos fallidos, control de acceso por rol y suspensión/reactivación de cuentas de cliente sin pérdida de datos clínicos.
  • Integraciones: Almacenamiento de adjuntos en Backblaze B2 (S3-compatible) con validación de tipos MIME y descarga segura desde la API.
  • Frontend: SPA en React 19 + TypeScript + Tailwind (agenda, historias clínicas, PDF de altas, panel de administración).
  • Despliegue: Docker Compose; Vercel (frontend), Render (backend) y Neon (PostgreSQL).
Spring Boot JWT PostgreSQL React TypeScript Docker Backblaze B2 Tailwind
